#  Adivina el Número - Android App

Una aplicación móvil interactiva desarrollada de forma nativa para Android. El objetivo del juego es simple: el usuario debe adivinar un número secreto generado aleatoriamente entre 0 y 100 en la menor cantidad de intentos posibles.

##  Tecnologías y Herramientas

*   **Lenguaje:** Kotlin
*   **UI Framework:** Jetpack Compose
*   **Diseño:** Interfaz moderna con diseño inspirado en Glassmorphism (paneles translúcidos y degradados).
*   **Librerías Adicionales:** `material-icons-extended` para la iconografía avanzada.
*   **IDE:** Android Studio

##  Características Principales

*   Generación de números aleatorios en tiempo real manteniendo el estado de la UI.
*   Validación estricta de entrada de usuario (uso de teclado numérico y bloqueo de caracteres no numéricos).
*   Feedback visual inmediato mediante tarjetas dinámicas e iconos de Material Design.
*   Manejo de estados con `remember` y `mutableStateOf` dentro del ecosistema de Compose.
*   Animaciones fluidas (`AnimatedVisibility`, `fadeIn`, `fadeOut`) para la aparición de resultados.
